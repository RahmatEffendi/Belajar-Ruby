Parameter Method

    Method telah dijelaskan pada pembahasan sebelumnya, pada hari ini akan dibahas tentang Parameter. Parameter merupakan
bagian terpenting dari method. Jika Parameter tidak dimiliki oleh method, method masih bisa berjalan, namun itu hanya
untuk penggunaan program sederhana saja. Namun bagaimana dengan pengolahan data skala besar misalnya mengolah data
pegawai perusahaan yang memiliki beberapa Parameter yang sama namun berbeda pegawai. Untuk itu Parameter membantu
developer dalam memudahkan proses pengcodingan.

Parameter itu sendiri merupakan item-item yang digunakan sesuai dengan kebutuhan yang diinginkan, dengan kata lain
parameter dibuat oleh seorang developer dalam melakukan suatu proses. Pada method parameter bisa digunakan lebih dari satu parameter atau satu saja parameter juga bisa, tergantung dari kebutuhan 
yang diinginkan. Perbandingannya dapat dilihat pada contoh program sederhana dibawah.

    parameter - parameter di atas bisa dipanggil tanpa harus berurutan dari nama sampai usia, bisa dimulai dari

    Agar parameter dapat dipanggil, disetiap parameter pada method di atas ditampahkan titik dua : , setelah
    tanda titik dua kemudian diikuti oleh parameter-parameter yang akan dipanggil, dapat dilihat pada contoh di atas
    ketika user telah memasukkan semua input yang diminta kemudian method dieksekusi pada parameter method dapat dilihat
    setiap parameter menggunakan titik dua dan nama dari parameternya, misalnya nama:nama. Parameter kedua dari nama
    bisa menggunakan kata sesuai keiinginan, misalnya nama:nm , Namun karakter nama diawal harus sesuai dengan parameter
    yang telah didefinisikan pada method.
=end

def data(nama_awal:, nama_akhir:, nama_alias:)

    puts "Alias : #{nama_alias}"
end
