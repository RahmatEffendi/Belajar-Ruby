    METHOD

    Method adalah suatu blok kode yang didalamnya berisikan tugas secara spesifik, yang akan digunakan
    atau bisa digunakan oleh kode lain dalam pemanggilannya di luar dari blok method tersebut. Method 
    sama hal nya dengan bahasa pemprograman pada umumnya fungsi dan kegunaannya juga sama, namun cara
    penulisannya saja yang berbeda karena ini adalah Ruby bukan PHP, python atau lainnnya.

    Aturan penamaan method pada ruby sama dengan aturan penamaan pada bahasa pemprograman lain pada
    umumnya, Namun pada ruby bisa diawali dengan underscore (_), tanda tanya (?) dan tanda seru (!).
    Method pada ruby dibagi menjadi dua bagian :
        1. Method dengan nilai balik,
        2. Method tanpa nilai balik

    source : https://github.com/RahmatEffendi/Belajar-Ruby/blob/master/method.rb

    1. Method dengan nilai balik
        Mehtod dengan metode ini akan mengembalikan nilai balik ketika method tersebut dieksekusi
    
    contoh :
        print "Masukkan Nilai : "
        nilai = gets.to_i
        puts nilai
    
    Program diatas akan meminta user untuk menginput nilai berapa pun jumlahnya. Ketika nilai sudah diinput
    maka number yang diinput akan dimasukkan ke object yang diberinama nilai. Pada object nilai ini memiliki
    method dengan fungsi nilai balik yaitu gets(), method gets akan mengembalikan value yang dimasukkan user
    dengan menggunakan keyboard. Setelah method gets() disisipkan method to_i(), berfungsi untuk menunjukkan 
    bahwa setiap nilai yang diinput akan bernilai integer (bulat) bukan float. Misalnya user menginputkan nilai
    2.333, maka yang akan dikembalikan adalah nilai 2 saja.

    2. Method tanpa nilai balik
        Method tanpa nilai balik tidak akan membalikkan nilai ketika method tersebut dieksekusi.

    contoh :
        def perkalian (x,y)
            puts x * y
        end
        #Pemanggilan method
        perkalian(2,3)
    
    Program diatas tidak akan mengembalikan nilai balik dari method nya sendiri. Kita lihat pada program diatas
    menggunakan method puts(). Pada method perkalian() diatas method puts() hanya berfungsi untuk menampilkan
    nilai kepada user saja.
